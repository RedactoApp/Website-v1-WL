FAQ

Common questions

How does Redacto hide windows without affecting your own view?

Redacto uses macOS's native screen capture APIs to intercept what gets shared with conferencing apps like Zoom, Google Meet, and Teams. It renders a replacement overlay frame where the sensitive window would appear — only in the stream, not on your actual display. You still see everything normally.

Which conferencing apps does it support?

Zoom, Google Meet, Microsoft Teams, Loom, and any app using macOS's standard screen capture APIs. If it works with macOS screen sharing, Redacto works with it.

Does this work for full-screen sharing and window-only sharing?

Yes — both whole-screen sharing and individual window sharing are supported. Tab-level redaction requires full-screen sharing mode.

Will this break any macOS permissions or policies?

No. Redacto uses only official macOS APIs and is fully notarized by Apple. It requests Accessibility and Screen Recording permissions, both of which you control and can revoke anytime.
